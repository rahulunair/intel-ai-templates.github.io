Deployment Steps

1 Create a standard account on Intel Tiber AI Cloud

Visit Intel Cloud Portal

2 Select an Intel Data Center Max Series GPU VM

3 Clone the repository 

git clone https://github.com/tiberaicommunity/xpu_tgi

4 Generate authentication token 

python utils/generate_token.py

5 Start the model 

./start.sh phi-3-mini-4k

6 Make a test request 

curl -X POST http://localhost:8000/generate \
    -H "Authorization: Bearer YOUR_TOKEN" \
    -H "Content-Type: application/json" \
    -d '{"inputs": "Hello, how are you?", "parameters": {"max_new_tokens": 50}}'
